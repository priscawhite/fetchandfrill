## Purpose of This Project
This project was created as part of a web development learning experience focused on strengthening front-end layout and styling skills using Flexbox. It combines creative branding, responsive design concepts, and modern UI practices into a polished company homepage project.

## Project Goals
In this project, flexbox was used to design and build the layout for a company homepage. The goal was to create a visually appealing, responsive, and modern website experience for a fictional company while practicing layout structure and styling techniques using CSS Flexbox.

## Project Requirements
This project includes the following:
* A title and logo/splash image
* A mission statement and company introduction
* Product sections featuring images and titles
* An employee/team section introducing company members  

Custom styling including:
* colors
* fonts
* borders
* spacing and padding
* Flexbox styling for responsive layouts and content organization

## Fetch & Frill
Fetch & Frill is a fictional upscale online boutique for dog lovers who appreciate modern style, thoughtful design, and fashionable pet accessories. Founded in Austin, Texas, the brand was created to bring trendy, curated products to pet parents while supporting animal shelters across the United States.

[Visit the site](https://priscawhite.github.io/fetchandfrill/)
